# Introduction

This is an example article written in Markdown. It's much easier than writing HTML!

You can write naturally with **bold text**, *italic text*, and [links](https://example.com).

## Adding Images

You can add images using standard Markdown syntax:

![Alt text for image](images/example.jpg)

Images will be automatically copied to the `media/articles/` directory when you create the article.

## Code Blocks

You can include code with syntax highlighting:

```python
def hello_world():
    print("Hello, world!")
```

## Lists

Unordered lists:
- Item one
- Item two
- Item three

Ordered lists:
1. First item
2. Second item
3. Third item

## Quotes

> This is a blockquote. It will be styled beautifully in your article.

## Formatting

You can use all standard Markdown features:

- **Bold text**
- *Italic text*
- `Inline code`
- ~~Strikethrough~~
